### 네이밍 컨벤션

1. **패키지명:**
   - **소문자**로 작성.
   - 각 기능별로 패키지를 나누어 관리.

2. **클래스명:**
   - **PascalCase**를 사용하여 작성.

3. **메소드명:**
   - **camelCase**를 사용하여 작성.
  
4. **파일명:**
   - 자바 파일과 리소스 파일 모두 클래스명 또는 설정명을 따름.
